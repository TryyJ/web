BO7 Ranked Play Meta Classes (Feb 2026)

CDL-restricted meta – only a few guns dominate high-level play. Updated from pro sources & trackers.

1. M15 Mod 0 (Best AR – Versatile / Lane Control)

Pick Rate: ~27-30% in ranked. Pro staple for mid-long range.

  • Optics: Lethal Tools ELO or Kepler Microflex
  • Barrel: 16.5” Fusion / Reinforced Barrel
  • Rear Grip: Peregrine / Ergonomic Grip
  • Stock: Wander 3V / Infiltrator Stock
  • Fire Mods: Buffer Spring / Recoil Springs

Loadout Code Example: A01-21WSG-MJ5UL-KB11 (or similar pro variants). Low recoil, great for holding hills/spawns.

2. Dravec 45 / Sturmwolf 45 (Top SMG – Aggro Rush)

Pick Rate: ~9-28% (aggressive close-range king).

  • Optic: Kepler Microflex (if allowed)
  • Barrel: Long / Reinforced
  • Rear Grip: Quickdraw / Ergonomic
  • Magazine: Fast Mag II
  • Underbarrel: Vertical Foregrip

Loadout Code Example: S07-D46TJ-111 or S04-CWWDU-H11. High mobility, fast TTK up close – pair with movement perks.

Meta shifts with patches (e.g., Season 2 tuning on recoil/attachments). Check CDL rules & trackers like wzstats.gg or warzoneloadout.games for latest. Last updated: Feb 16, 2026.
